# GitHub Octoverse 2026: The State of Open Source and Agentic Workflows
Source: GitHub, 2026
URL: https://octoverse.github.com/2026

Summary: The 2026 Octoverse report documents a 40% rise in AI-assisted commits across open source repositories. It also reports a corresponding increase in reverting AI-generated code due to subtle bugs and security flaws, reinforcing the need for human code review, security auditing, and ethical oversight as core engineering skills.
